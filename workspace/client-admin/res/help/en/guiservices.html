This panel allows the administrator to add new Gui services in SITools2. Once a Gui service is defined in the system, the user can use it on the dataset views

Create a Gui service

  1. Click on Create
  2. Fill in the name, description, xtype of javascript object, image css class, priority and all other Gui service parameters
  3. Fill in the dependencies in the second tab (dependencies can only be js or css file urls)
  4. Click on Ok

Edit a Gui service

  1. Select a Gui service  in the list
  2. Click on Edit.
  3. update the name, description, xtype, icon css class priority or any other properties
  4. Click on Ok

Delete a Gui service

  1. Select a Gui service  in the list
  2. Click on Delete
