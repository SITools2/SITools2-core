1.1 The portal (UG)

Le portail donne accès aux diffèrents projets.

Figure 1: Portail utilisateur

Il s'agit d'un portail contenant plusieurs «  portlets  » que l'on peut déplacer sur la page. Il est possible de choisir quels «  portlets  » sont affichés en les cochant ou en les décochant sur la partie Components à gauche. On peut consulter la liste des projets dans le «  portlets  » Projects ainsi que différents flux RSS définis au niveau du portail dans le portlet Portal feeds. On accède à un projet en double cliquant dessus.

1.2 Le bureau utilisateur

Figure 2: Bureau utilisateur

Ce bureau utilisateur permet d’accéder à différentes fenêtres de l’application. Ces fenêtres sont accessibles via le menu Démarrer ou via le « fisheye » présent au dessus de la barre du bureau.

1.2.1 Sauvegarde du bureau

Il est possible de sauvegarder son bureau (position des fenêtres disponible dans le menu Démarer) afin qu’elles s’ouvrent automatiquement lors d’une prochaine connexion. Pour cela il suffit de s’enregistrer, de posséder un espace utilisateur et de cliquer sur le bouton Sauvegarder dans le menu Démarer.


1.2.2 Explorateur de jeux de données

.

Figure 3: Dataset Explorer

Il permet d'explorer l'ensemble des jeux de données du projet et donne accès aux différents services sur ces jeux de donnèes. Pour accéder à un service il suffit de cliquer sur l'un d'entre eux.

1.2.3 Espace personnel

Figure 4: Visualisation du panier

Figure 5: Visualisation des préférences

Cette fenêtre permet de consulter son panier et les commandes effectuées. On peut également consulter ses préférences et les supprimer.

1.2.3.1 Consulter son panier

  1. Cliquer sur Organiser les commandes puis sur Commander
  2. Choisir le fichier à commander
  3. Cliquer sur Envoyer la commande pour effectuer sa commande.

1.2.3.2 Consulter ses commandes :

  1. Cliquer sur Organiser les commandes puis sur Voir les commandes
  2. Choisir sa commande
  3. Cliquer sur Détails pour voir le détail de sa commande
  4. Cliquer sur Supprimer pour supprimer sa commande

1.2.3.3 Consulter son espace utilisateur :

  1. Cliquer sur Préférences
  2. Déplier l'arbre

1.2.3.4 Supprimer un fichier de son espace utilisateur :

  1. Cliquer sur Préférences
  2. Déplier l'arbre
  3. Choisir le fichier à supprimer
  4. Cliquer sur Supprimer les ressources

1.2.4 Graphe du projet

Figure 6: Graphe du projet

Il permet d'accéder aux mêmes services sur les jeux de données que l'explorateur de jeux de données mais organisés d'une façon différente. Il offre également un peu plus d'informations, à savoir :

  • Le nombre d'enregistrements d'un jeu de données
  • L'image du jeu de données
  • Une description en format HTML du jeu de données.

1.2.5 Flux de projet

1.2.5.1 Consulter un flux

  1. Choisir un flux dans la liste des flux (ici : Nouvelles projet fuse)
  2. Consulter le flux

Sous Mozilla Firefox, il est également possible de s'abonner à ces flux en cliquant sur l'icône RSS à droite de la barre d'adresse.

1.2.6 Gestionnaire de bureau

Figure 7: Gestionnaire de bureau

Cette fenêtre permet de choisir les applications disponibles sur le bureau.

1.2.6.1 Ajouter une application

  1. Choisir une application dans la liste du bas
  2. Cliquer sur Ajouter au bureau (une seule application d'un type donné est affichable sur le bureau)

1.2.6.2 Supprimer une application

  1. Choisir une application dans la liste du haut
  2. Cliquer sur Supprimer du bureau

1.3 Services sur les jeux de données

1.3.1 Visualisation des données

Figure 8: Afficher les données

Cette fenêtre est accessible via Afficher les données de l'explorateur de jeux de données.

Elle affiche l'ensemble des données contenues dans le jeu de données.

On peut appliquer des services sur ces données :

1.3.1.1 Ajouter à mes sélections

  1. Sélectionner une ou plusieurs lignes (utiliser shift et ctrl pour en sélectionner plusieurs)
  2. Cliquer droit sur une des lignes
  3. Cliquer sur Ajouter à mes sélections

1.3.1.2 Ajouter toutes les données à ma sélection

  1. Cliquer droit sur une des lignes
  2. Cliquer sur Télécharger le jeu de données

1.3.1.3 Consulter les détails d'une donnée

  1. Avec menu contextuel
    1. Sélectionner une ligne
    2. Cliquer droit sur cette ligne
    3. Cliquer sur Détails
  2. Sans menu contextuel
    1. Double cliquer sur la ligne

1.3.1.4 Sauvegarde des paramètres de la fenêtre

Il est possible de sauvegarder les paramètres de cette fenêtre en cliquant sur Enr. configuration. Dans ce cas, sont sauvegardés la position et la taille de la fenétre, la position et la taille des colonnes ainsi que leurs visibilitées.

1.3.6 Recherche OpenSearch

Figure 13: Recherche OpenSearch

Cette fenêtre est accessible via le menu OpenSearch de l'explorateur de jeux de données.

1.3.6.1 Effectuer une recherche

  1. Saisir la recherche dans le champ Recherche
    1. Un mécanisme d'auto complétion permet d'être guidé pour la requête. Est affiché le terme, la colonne qui contient le terme ainsi que le nombre de résultats dans cette colonne.

    1. La syntaxe utilisée est la syntaxe OpenSearch utilisée par « Apache Lucene ».
  • Recherche sur une colonne particulière : nomColonne : recherche
  • Récupération de tous les résultats : * :*
  • Méta caractères : * pour un ensemble de lettre, ? pour une seule lettre (ne peut pas être utilisé comme premier caractère.
  • Plus d'informations sur la syntaxe Lucene :

http://lucene.apache.org/java/2_3_2/queryparsersyntax.html

  • Plus d'informations sur la syntaxe OpenSearch :

http://www.opensearch.org/Specifications/OpenSearch/1.1#Template_grammar

  1. Cliquer sur le bouton Recherche

1.3.6.2 Consulter les détails d'une donnée

Double cliquer sur la ligne de la donnée.

1.3.6.3 Sauvegarde des paramètres de la fenêtre

Il est possible de sauvegarder les paramètres de cette fenêtre en cliquant sur Enr. configuration. Dans ce cas, sont sauvegardés la position et la taille de la fenêtre.

1.3.6.4 Ajouter l'index à ses moteurs de recherche

Sous Mozilla Firefox, il est possible d'utiliser cet index OpenSearch comme moteur de recherche en l'ajoutant à la liste des moteurs de recherche

  1. Cliquer sur l'icône à gauche de la barre des moteurs de recherche
  2. Choisir l'index à ajouter (ici : fuse proposals )
  3. Effectuer ses recherches avec la même syntaxe OpenSearch.
